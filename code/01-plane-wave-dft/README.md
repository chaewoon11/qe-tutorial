# Chapter 1 — plane-wave basis size data

`npw_sweep.dat` — number of plane waves (smooth-grid G-vectors) in the GaAs
calculation as `ecutwfc` is varied from 20 to 80 Ry (all other settings as in
`../00-first-scf/inputs/gaas.scf.in`). Generated by running that SCF input with
each cutoff and grepping the `Smooth grid: NNNN G-vectors` line.

Used by `../../notebooks/01-basis-size.ipynb` to show that the basis size grows
as `ecutwfc^(3/2)`. The deeper analysis (number of k-points, symmetry, the two
FFT grids) reads directly from `../00-first-scf/ref/gaas.scf.out`.
